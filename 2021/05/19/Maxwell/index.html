Maxwell-mysql binlog 分析工具

step1.安装版本

jdk-8u231-linux-x64.tar.gz 略
kafka_2.12-1.0.0.tgz
mariadb-10.1.8.tar.gz
maxwell-1.25.0.tar.gz

step2.安装maxwell

tar -zxvf maxwell-1.25.0.tar.gz -C /home
cat config.properties |grep -v ‘#’ |grep -v ‘^$’

启动

启动可提前在kafka新建与配置文件同名topic，也可通过启动配置文件创建topic
./bin/maxwell –config /home/maxwell-1.25.0/config.properties

测试

kafka创建topic
./kafka-topics.sh –create –zookeeper 10.10.253.13:2181,10.10.253.14:2181,10.10.253.15:2181 –replication-factor 3 –partitions 1 –topic maxwell-test

mysql创建库表，插入信息测试
create database test；
create table xyzq(id int not null primary key,name varchar(20),age int,address varchar(20));
insert into xyzq values(4,’xyqaw’,0,’beijing’);

kafka 查看maxwell topic的输出结果
./kafka-console-consumer.sh –bootstrap-server 10.10.253.13:9092,10.10.253.14:9092,10.10.253.15:9092 –topic maxwell-test
